The AI Landscape

A one-page map for placing any AI term, technology, or job title. Keep it beside you; it stays true across the whole roadmap.

1The nesting

These four are Russian dolls, not rivals. Each sits inside the one before it.

ARTIFICIAL INTELLIGENCE — machines doing tasks that seem to need human intellect
MACHINE LEARNING — systems that learn patterns from data
DEEP LEARNING — multi-layer neural networks that learn features on their own
GENERATIVE AI / LLMs — models that generate new text, images, audio, code
LayerOne-line definitionExample
AIAny technique that makes a machine act "intelligently."A chess engine; a spam filter
Machine LearningAI that learns from data instead of being hand-coded with rules.Predicting house prices from past sales
Deep LearningML using many-layered neural networks that discover features themselves.Image recognition; speech-to-text
Generative AIDeep learning (usually transformers) that produces new content.ChatGPT, Claude, Midjourney

2Two neighbours that are not layers

  • Data Science — extracting insight and answers from data (stats, visualisation, some ML). Overlaps ML but aims at understanding, not shipping a product.
  • Data Engineering — building the pipelines, storage, and infrastructure that feed data to everything above. Plumbing, not modelling.

3Levels of AI

LevelMeaningStatus today
Narrow (ANI)Good at one task or a narrow band of tasks.Everything that exists — including today's LLMs
General (AGI)Human-level competence across any task.Hypothetical; not achieved
Super (ASI)Beyond the best humans at essentially everything.Speculative

"Generative" is a capability, not a level. A generative model can still be narrow — and today's are.

5The AI lifecycle (any project)

problem → data → pick or train a model → evaluate → deploy → monitor & iterate

As an AI Engineer you usually pick a pretrained model rather than train one — so your time goes into everything around the model.
