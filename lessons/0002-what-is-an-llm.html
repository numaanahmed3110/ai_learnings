What Is an LLM, Really?

One mechanism sits underneath ChatGPT, Claude, Gemini, and every tool you'll build. Grasp it and prompting, hallucination, and "context" all stop being mysterious.

Your win

You'll be able to explain what a Large Language Model fundamentally does in one sentence — and use that to predict why it hallucinates, why prompting works, and why it still counts as "narrow" AI.

1The one-sentence truth

Strip away the marketing and a Large Language Model (LLM) is a machine that does exactly one thing:

The whole trick

Given some text, it predicts the next chunk of text — then adds that chunk to the input and predicts again. And again. That loop is the entire show.1

Those "chunks" are called tokens (roughly, word-pieces — we'll go deep on them next lesson). So more precisely: an LLM looks at the tokens so far and produces a probability for every possible next token — "the" 8%, "a" 5%, "cat" 0.1%, … — then one is picked, appended, and the process repeats. Generating a paragraph is just this step run hundreds of times. This repeated self-feeding is called autoregression.2

"The cat sat on the ___"
model reads the tokens → ranks every possible next token → picks "mat" → feeds it back → repeats

2So where does the "intelligence" come from?

Here's the surprising part. To predict the next token well across billions of sentences — code, essays, dialogue, math — the model is forced to internalise grammar, facts, styles, and reasoning patterns. Good prediction requires a rich model of the world. Trained on a huge slice of the internet, these abilities emerge from that one simple objective.1 Nobody hand-coded "grammar" — it fell out of relentless next-token practice.

3Three things this instantly explains

PuzzleExplained by "it predicts the next token"
Why it hallucinates It generates plausible-sounding text, not verified facts. By default it isn't looking anything up — so it can produce confident, fluent, wrong answers.1
Why prompting works Your prompt is the text it predicts from. Better, clearer context steers the probabilities toward the answer you want.
Why it's still narrow It's a spectacular language pattern-predictor, not a general mind with goals or understanding. (Recall the levels from Lesson 01.)
Mental model to keep

An LLM is autocomplete on steroids — a very, very good next-token predictor. Powerful, but it has no built-in notion of "true." That single fact will shape almost every engineering decision you make later (evaluation, RAG, guardrails).

4One nuance: base vs chat models

A raw next-token predictor (a base model) just continues text — hand it "Once upon a" and it rambles onward. The helpful, instruction- following assistant you talk to (a chat / instruct model) is that base model further shaped to follow instructions and stay helpful. How that shaping works — pre-training, fine-tuning, RLHF — is a later lesson; for now just know the assistant is a tamed version of the raw predictor.

Recall (say it before revealing): what is an LLM, in one sentence?

A Large Language Model is a system that, given some text, repeatedly predicts the most likely next token — and from that one objective, grammar, facts, and reasoning patterns emerge. It generates plausible language; it does not look up truth.
